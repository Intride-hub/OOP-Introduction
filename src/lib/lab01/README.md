## Разработка класса car

### Атрибуты
В первую очередь определяем ключевые характеристики машины

- `_brand` - марка
- `_speed` - скорость
- `_fuel` - количество топлива
- `_capacity` - вместимость

**атрибут класса** 
- Transport - характеристика для каджого экземпляра класса

### Свойства `@property`

- `brand` - получение марки машины (чтение)
- `speed` - получение скорости (чтение, setter с валидацией для изменения)
- `capacity` - получение вместимости(чтение)
- `fuel` - получение топлива(чтение)


**Инкапсуляция**
- Закрытые поля в атрибутах(с префиксом _) чтобы избеежать прямого изменения атрибутов
- Доступ через @property
- изменение скорости(speed) через setter с валидацией

---

### Валидация

- `_validate_brand` - проверка марки(строка, не пустая)
- `_validate_speed` - проверка скорости(число, >0)
- `_validate_capacity` - проверка вместимости(число, >0)
- `_validate_fuel` - проверка топлива(число, >0)

---

**Магические методы**

* str - строковое представление для пользователя
* repr - техническое представление объекта
* eq - сравнение обьектов (по скорости и марки машины)


---


**Бизнес логика**


три простых, базовых методов

- `start_engine()` - включает двигатель(если топлива < 0 выбрасывает ошибку)
- `stop_engine()` - выключает двигатель
- `drive()` - езда (при отключенном двигателе выбрасывает ошибку, расход топлива 10км/1л, при нехватке топлива ошибка)
