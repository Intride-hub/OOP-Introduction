# Разработка класса Fleet


### Задание на 3

* Реализован класс Fleet, который хранит обьекты класса Car

#### Методы 
* `add()` - добавляет обьект в коллекцию (проверка на тип обьекта и дубликат)
* `remove()` - удаляет обьект из коллекции
* `get_all()` - возвращает список обьектов


### Задание на 4
* `find_by_id` - поиск по id машины
* `find_by_brand` - поиск по марке машины(с приведением к строчному виду)
* `__len__` - возвращает количество элементов в коллекции
* `__iter__` - позволяет перебирать коллекцию в цикле

 ### Задание на 5 
 * `__getitem__` - доступ по индексу
 * `remove_at(index)` - удаление по индексу
 * `sort_by_speed` - сортировка по скорости
 * `sort_by_brand` - сортировка по марке(по алфавиту)
 * `get_cars_with_engine_on` - возвращает коллекцию машин с включенным двигателем
 * `get_cars_with_low_fuel` - возвращает коллекцию машин с топливом меньше определенного порога
